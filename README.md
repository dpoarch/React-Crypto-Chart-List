## React Coin Ranking

Runs with NodeJS backend which scrape top 10 coins on `coinmarketcap` and fetch api with `coingecko`


## Getting Started
Run the following command to setup the project:

```
npm install
```

```
npm run start
```


## NodeJS URL
```
http://localhost:4000
```

## ReactJS URL
```
http://localhost:3000
```
